ANTP - Associação Nacional de Transportes Públicos

Duração do projeto: 3 meses

Data Início: 2012 - Dezembro

Data Término: 2013 - Fevereiro

Website: antp.org.br

The Client

A ANTP é uma entidade civil, sem fins lucrativos, criada em 1977, voltada ao setor de transporte público e do trânsito do Brasil e que tem por objetivo desenvolver e difundir conhecimentos visando seu contínuo aprimoramento.

The Challenge

A ANTP possui diversos conteúdos publicados em diferentes websites(hotsites) que foi necessário traze-los e organizá-los tudo dentro do novo Portal

The Solution

A solução foi coletar todos os materiais das diversas fontes, e processar cada conteúdo migrando para o novo formato de armazenamento do novo portal.

Como foi o projeto

Parte do projeto foi executado durante a reconstrução do novo portal, prevendo como os conteúdos seriam reestruturados para serem acomodados dentro do portal.

Então nesta etapa, meu trabalho foi coletar todos os materiais existentes, convertendo formatos de dados, extraíndo conteúdos em arquivos/documentos, reoganizando bancos de dados, etc.

Para então transferir para o novo modelo.

Project Results

Liberamos os conteúdos reorganizados dentro do novo portal à medida que fomos concluíndo cada seção/tipo de material.

Conteúdos de notícias, agenda de eventos, biblioteca, sub-sites/hot-sites de eventos, projetos, ...

Após 3 meses, concluímos a migração com praticamente 100% do conteúdo organizado na nova plataforma.
